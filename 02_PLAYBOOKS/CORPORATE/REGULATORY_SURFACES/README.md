# Regulatory Surfaces

Mapping of regulatory touchpoints for corporate transactions.

---

## Purpose

Document which regulators, filings, and compliance obligations attach to specific:
- Transaction types
- Entity structures
- Industry sectors

---

```markdown
## Applicable Regimes

- Industry-specific

| Filing | Authority | Timing | Consequence of Miss |
|--------|-----------|--------|---------------------|
| | | | |

## Ongoing Obligations
- [ ] Obligation 1
- [ ] Obligation 2

## Common Pitfalls
- Pitfall 1
- Pitfall 2
```
